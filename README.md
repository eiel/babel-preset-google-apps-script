# babel-preset-google-apps-script

> Babel preset for Google Apps Script

## install

```sh
npm install --save-dev babel-preset-google-apps-script
```

## Usage

### Via `.babelrc` (Recommended)

**.babelrc**

```json
{
  "presets": ["google-apps-script"]
}
```
